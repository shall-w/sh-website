Korg Volca Beats Sample Pack Download

October 11, 2017

Hi everyone,

A few weeks ago I posted on /r/technoproduction asking if anyone would be interested in a Korg Volca Beats sample pack, geared towards techno music. I got a few “yes, obviously” responses haha… so here it is!

Where demo songs are featured on the page, the sounds have been dropped into a sampler, tweaked in pitch/volume, had a little delay, reverb and/or Ableton limiter added, and that’s all. All the sounds in these songs come from the pack, definitely take a listen as I think they are a good example of what you can throw together with these sounds!

The Sounds

Set 1 — Dry:
First of all, the pack contains a set of 47 drum sounds straight from the Volca. Not only does the pack contain the standard expected sample variations (long open hat, short open hat; punchy kick, soft kick etc.), you’ll also find some sounds labelled “crunchy”. These have been slightly mashed up in some way using the Korg’s built-in “Stutter” feature, giving them a sort of grainy, lo-fi vibe.

Here’s a demo of all the Dry folder’s sounds:

Shallowed · Dry - Raw Sounds

As you might notice, the sounds within this initial set lend themselves to a minimal vibe. Here’s a little demo song using them:

Shallowed · Dry - Demo

Set 2 — Processed1:

As well as these sounds straight from the machine, there are two other folders, each containing a processed set of the same 47 drum sounds.

The samples found in Processed1 have been run through a tape saturation plug-in. The effect is quite subtle, but the resulting drum sounds lend themselves to raw, lo-fi style tracks.

Here’s the raw demo of Processed1’s sounds:

Shallowed · Processed 1 - Raw Sounds

Here’s a demo song using the sounds from Processed1:

Shallowed · Processed 1 - Demo

Set 2 — Processed2:

The samples found in Processed2 have been smashed through a digital distortion plug-in. Expect fat, blown-out kicks and noisy, crackly hi-hats in this batch of sounds. Good for wonky techno among other things…

License

Here’s a little license I threw together for these, I wanted to be clear for your sake that you’re free to use these pretty much however you like in your music. The only thing I am keen to avoid was someone trying to sell the samples as part of another sample pack, or something like that (as unlikely as that may seem!).

Here’s the full license that’s included with the pack:

• Use these samples as part of a (commercial or non-commercial) music production in any way you see fit, without attribution. No further permission is required from me (the creator of these samples) if you wish to use these as part of a commercial track. You are absolutely allowed to use these in your music.

• Distribute these samples as-is, for free, to anyone else who might make use of them. Please also pass on this license if doing so :) If you wish to distribute this whole collection as part of of a free sample pack collection (i.e. on a web forum), that is also fine.
